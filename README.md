# 🚀 FanFund – Empower Creators Through Fan Support

**FanFund** is a modern web platform that enables creators to receive donations directly from their fans. Built using **Next.js**, **Tailwind CSS**, and **Razorpay**, the app provides a smooth, secure, and interactive donation experience — from login to checkout to success confirmation.

---


## ✨ Features

- 🔐 **Login & Dashboard**: Personalized welcome with secure access.
- 💸 **Razorpay Integration**: Accept real-time donations with payment success confirmation.
- 🎯 **Dynamic Amounts**: Accept donations through custom amount passed via query (`/pay?amount=200`).
- 🎨 **Responsive UI**: Built using Tailwind CSS for a smooth modern user experience.
- ⛔ **Back Navigation Blocked**: Once redirected to the success page, users can’t go back.
- 📲 **Social Links**: Connect with the creator on GitHub, Instagram, Twitter, and LinkedIn.

---

## 🛠 Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Payments**: [Razorpay Checkout.js](https://razorpay.com/docs/payment-gateway/)
- **Auth**: [NextAuth.js](https://next-auth.js.org/) 

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Sambhav1401/FanFund.git
cd FanFund
npm install
npm run dev
Then open http://localhost:3000 in your browser.

```
